Notes (click to expand)

Currently works with videos, images, and audio. All processing is done 100% client-side.

Videos, especially longer ones, may take some time to process. Please be patient.

Files containing opt_ in their names will be skipped by the optimization algorithm but will still be passed as a download if selected.

opt_ will be suffixed to the existing filename for each file that has been successfully processed. Files that already have opt_ in their names and have been skipped, will not receive it a second time.
